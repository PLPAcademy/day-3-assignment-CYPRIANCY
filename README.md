#Dart & Flutter

1. What is the first step for installing Dart on a Windows machine?

A) Install Homebrew
B) Download the Dart SDK
C) Update your PATH
D) Run Dart Doctor

ANWSER: 
(A) Download the Dart SDK (This is the first step because you need the actual software before configuring your system)

2. Which command verifies the Dart installation on macOS?

A) dart --install
B) dart --check
C) dart --verify
D) dart --version

ANWSEER: 
(D) dart --version (This command checks the installed version of Dart)

3. What is the next step after downloading and extracting the Flutter SDK on Linux?

A) Install Homebrew
B) Update your PATH
C) Run Flutter Doctor
D) Create a new Flutter project

ANWSER: 
(B) Update your PATH (This allows you to run Flutter commands from anywhere in the terminal)


4. What command is used to run a newly created Flutter app?

A) flutter start
B) flutter build
C) flutter run
D) flutter init

ANWSEER: 
(C) flutter run (This command starts a development version of your Flutter app on a connected device or emulator)



#Python Installation

What is the first step to install Python on a Windows system?

A) Run the installer without any customization
B) Download Python from the official website
C) Open the terminal and type sudo apt install python
D) Install pip manually

ANWSER: 
(B) Download Python from the official website (This is the official and secure way to get Python)

Which option should you ensure is checked during Python installation on Windows?

A) Install with default settings
B) Install to a custom directory
C) Add Python to PATH
D) Install all available features

ANWSER: 
(C) Add Python to PATH (This ensures you can run Python commands from anywhere in the terminal)

How do you verify Python installation on any system?

A) By running python --version
B) By restarting your computer
C) By opening the Python installer again
D) By checking the Programs and Features in Control Panel

ANWSER: 
(A) By running python --version (This is the standard way to check if Python is installed and its version)

What command is used to install pip on macOS and Linux?

A) sudo install pip
B) pip install python
C) curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
D) python --install pip

ANWSER: 
(C) curl 

https://bootstrap.pypa.io/get-pip.py -o get-pip.py (This command downloads the script to install pip on macOS and Linux)

What is the purpose of a virtual environment in Python?

A) To keep your projects organized in one folder
B) To install Python in a different directory
C) To separate project dependencies and avoid conflicts
D) To enhance Python performance on your system

ANWSER: 
(C) To separate project dependencies and avoid conflicts (Virtual environments create isolated environments for each project, preventing conflicts between project-specific libraries)

#MySQL Installation

What is the first step to install MySQL on Windows?

A) Download MySQL Shell
B) Download MySQL Installer from the official website
C) Install MySQL Workbench
D) Set up a root password

ANWSER: 
(B) Download MySQL Installer from the official website (This is the official and secure way to get MySQL)

What setup type should you choose for a custom MySQL installation?

A) Developer Default
B) Server Only
C) Full
D) Custom

ANSWSER: 
(D) Custom (This allows you to choose specific components you need)

Which products should you select during the MySQL installation?

A) MySQL Server, MySQL Workbench, and MySQL Shell
B) Only MySQL Server
C) MySQL Server and MySQL Router
D) MySQL Workbench and MySQL Utilities

ANWSER: 
(A) MySQL Server, MySQL Workbench, and MySQL Shell

What is the purpose of setting a root password during MySQL installation?

A) To create a user account for your MySQL server
B) To secure your MySQL installation with a super-secret password
C) To activate MySQL Workbench
D) To allow multiple users to access MySQL

ANWSER: 
(B) To secure your MySQL installation with a super-secret password (The root user has full access to the database and needs a strong password)

How do you begin managing your database after installing MySQL?

A) Start by installing additional plugins
B) Launch MySQL Workbench and connect to your MySQL Server
C) Run mysqladmin start in the terminal
D) Restart your computer to activate MySQL

ANWSER: 
(C) Launch MySQL Workbench and connect to your MySQL Server (This is the graphical interface to manage your database).
